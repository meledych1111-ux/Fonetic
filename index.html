Главное 📖 Смысл + Звук
妈 mā = 女(женщина) + 马(лошадь)
码 mǎ = 石(камень) + 马
蚂 mǎ = 虫(насекомое) + 马
骂 mà = 吅(кричать) + 马
💡 Видишь 马? Звук ma! 马 mǎ (лошадь) даёт звуковую основу. Тон может меняться, но база ma остаётся.

Почему это работает? Более 80% китайских иероглифов — фоноидеограммы (形声字). Один компонент отвечает за смысл (радикал), другой — за произношение (фонетик).

🎯 Ключ-смысловик (радикал):
氵 (вода) → 河 hé (река), 海 hǎi (море), 湖 hú (озеро), 洗 xǐ (мыть). Видишь три капли? Значит, связано с водой.
🎵 Ключ-фонетик:
青 qīng → 请 qǐng (просить), 清 qīng (чистый), 晴 qíng (солнечно), 情 qíng (чувство). Один звук — много иероглифов.
📌 Золотое правило мнемоники: Найди внутри сложного иероглифа знакомый простой — скорее всего, он подскажет чтение.
例: 跑 pǎo (бежать) = 足 (нога) + 包 bāo (звук). 抱 bào (обнимать) = 扌 (рука) + 包 bāo. Видишь 包? Звук bao!
Секрет 1 🎵 Ключ-фонетик: один элемент → одинаковое звучание

Главная идея: Если ты видишь знакомый иероглиф внутри другого — скорее всего, он подсказывает произношение. Это самый мощный инструмент для быстрого набора словарного запаса.

🌟 Звездный пример: 青 qīng (зелёный/синий, чистый)
请 qǐng (просить) = 讠(речь) + 青
清 qīng (чистый) = 氵(вода) + 青
晴 qíng (солнечно) = 日(солнце) + 青
情 qíng (чувство) = 忄(сердце) + 青
睛 jīng (глазное яблоко) = 目(глаз) + 青
蜻 qīng (стрекоза) = 虫(насекомое) + 青
💡 Запомни: Все они звучат как qing или jing! Тон может меняться, но каркас остаётся.

Главная идея: Всего 10 ключей покрывают половину всех иероглифов в повседневном использовании. Выучи их — и ты будешь узнавать структуру большинства слов.
